# Student Expense Tracker

A web-based expense tracking application developed using Python Flask and MySQL. 
This project helps students manage and monitor their daily expenses efficiently.

## Features
- Add new expenses with title, amount, category and date
- View all expenses in a structured table
- Delete expenses when needed
- Category wise expense organization

## Technologies Used
- Python
- Flask
- MySQL
- HTML
- Git & GitHub

## Database
- Database Name: expense_tracker
- Table: expenses

## How to Run
1. Install Flask: pip install flask
2. Install Flask MySQL: pip install flask-mysqldb
3. Start WAMP and create database expense_tracker
4. Run the app: python app.py
5. Open browser and go to: localhost:5000
